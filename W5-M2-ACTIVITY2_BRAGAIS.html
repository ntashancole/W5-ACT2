RRed is considered to be a color of intense emotions, ranging from anger, sacrifice, danger, and heat, through to love, passion, and sexuality. Its hex code is #FF0000.
OOrange is the color of the tasty citrus fruit. It is a secondary color and the result of mixing red and yellow in equal portions. The hex code for orange is #FFA500.
YYellow sits between yellow-orange and yellow-green in the color wheel. Being the most visible color from a distance, it is often used as a cautionary color. The hex code for yellow is #FFFF00.
GGreen is the result of mixing blue and yellow. The many variations of green depend on the proportions of the two primary colors. The green hex code is #00FF00.
BBlue is a color that has long been associated with royalty, art, military, business, and nature. The blue hex code is #0000FF.
IIndigo, found between blue and violet in the color wheel, is a natural pigment that comes from plants and is traditionally considered one of the 7 major spectral colors. The indigo hex code is #4B0082.
VViolet, named after the flower whose shade it bears, stands alone at the end of the visible light spectrum and sits between the wavelengths of invisible ultraviolet light and blue light. The hex code for violet is #8F00FF.
